# Estudos em Julia

Este repositório contém exemplos e estudos básicos da linguagem de programação Julia. Cada arquivo demonstra conceitos fundamentais e operações comuns na linguagem. Abaixo está uma breve descrição de cada arquivo:

## Arquivos

- **`hello-world.jl`**: Um exemplo simples de como imprimir "Hello, World!" em Julia.
- **`variaveis.jl`**: Demonstração de como declarar e usar variáveis em Julia.
- **`operacao.aritmetica.jl`**: Exemplos de operações aritméticas básicas, como soma, subtração, multiplicação, divisão, módulo e exponenciação.
- **`numeros-complexos.jl`**: Trabalhando com números complexos em Julia.
- **`funcoes1.jl`**: Introdução à criação e uso de funções.
- **`controle-fluxo.jl`**: Exemplos de estruturas de controle de fluxo, como `if`, `else` e `while`.
- **`escopolocal.jl`**: Explicação e exemplos de escopo local em Julia.
- **`escopoglobal.jl`**: Explicação e exemplos de escopo global em Julia.
- **`variaveisdeescopo.jl`**: Comparação entre variáveis de escopo local e global.
- **`condicional.jl`**: Exemplos de uso de estruturas condicionais.

## Sobre Julia

Julia é uma linguagem de programação de alto desempenho para computação técnica. Ela combina a facilidade de uso de linguagens dinâmicas com a velocidade de linguagens compiladas. Julia é amplamente utilizada em áreas como ciência de dados, aprendizado de máquina, simulações científicas e muito mais.

## Como executar os exemplos

1. Certifique-se de ter o Julia instalado em sua máquina. Você pode baixá-lo em [https://julialang.org/downloads/](https://julialang.org/downloads/).
2. Navegue até o diretório onde os arquivos `.jl` estão localizados.
3. Execute um arquivo usando o comando:
   ```
   julia nome-do-arquivo.jl
   ```
